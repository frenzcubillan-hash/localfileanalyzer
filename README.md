# 🤖 AI File Analyzer Chat

I made this because I was bored and like lazy to analyze my files one by one.


An intelligent AI-powered chat application that allows users to upload documents, images, and files, then interact with an AI assistant that can analyze, summarize, explain, and answer questions about the uploaded content. The system maintains chat history, supports multiple conversations, and provides a modern ChatGPT-style interface powered by local AI models.

# 🚀 Project Highlights

### 🧠 Learned Retrieval-Augmented Generation (RAG)
Implemented a system where uploaded documents are converted into usable AI context, allowing the assistant to answer questions based on file contents instead of relying only on model knowledge.

### 📂 Learned File Processing
Worked with multiple file formats such as PDF, DOCX, and images while extracting useful information for AI analysis.

### 🗄️ Learned Database Design
Designed and managed relational database structures for chats, messages, files, and event tracking using MySQL.

### ⚡ Learned Real-Time AI Streaming
Implemented streaming responses from Ollama so users can see AI responses being generated live.

### 📱 Learned Responsive Web Development
Built a user interface that works across desktop and mobile devices while maintaining usability and accessibility.

### 🔗 Learned API Development
Created backend API routes for chat management, file uploads, AI processing, and database communication.

### 🤖 Learned Local LLM Deployment
Integrated locally hosted language models through Ollama, allowing AI processing without relying on cloud-based APIs.
